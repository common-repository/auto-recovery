Allow your website to be recovered if you're no longer able to access it


== Description ==

Plugin is in beta, so please let us know if you run in to any problems in the support forum!

Automatically send a reminder or create a new admin account sent to a specific email address if the admin area of your site hasn't been accessed for a certain amount of time, once options have been enabled.


== Installation ==

1. Upload the "auto-recovery" folder to the "/wp-content/plugins/" directory
2. Activate the plugin through the "Plugins" menu in WordPress
3. Use the newly created admin menu to create recovery notifications and alter settings

== Frequently Asked Questions ==

= What's the point of the plugin? =

The purpose of this plugin is to allow someone you trust to continue your site, even if you're no longer able to.
